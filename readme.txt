== Description ==

Change Mail Sender is a very simple plugin to change the sender name in WordPress outgoing emails. It takes Automatically site name in emails.

No coding required.

Simply download Change Mail Sender plugin, activate it.


== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. change will be seen in email sender name

== Frequently Asked Questions ==

 How can I use this plugin?
 You can download it and active it. Sender name will come in emails.
